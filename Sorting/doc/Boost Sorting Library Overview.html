Introduction

This library contains the sorting functions Quicksort, Mergesort, Radixsort, and Radix Quicksort, and an interface class sortable. See the documentation of individual header files for more information. Each function except for radix sort has a reverse version, have the same name with a r_ at the front. So, quicksort would become r_quicksort.
