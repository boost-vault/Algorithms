Introduction

Mergesort is a medium speed sorting algorithm that uses comparisons to sort objects. Unlike quicksort, it is stable, and does not have a worst case, but it is not as fast as best-case quicksort.

Functions 
template<class T> void mergesort(T* a, int size)
Requires: that for any object t of class T, t < other_t must be defined. 
Effects: that a is sorted from highest to lowest up to size
Complexity: best and worst case: O(N log N) where N is the number of elements
